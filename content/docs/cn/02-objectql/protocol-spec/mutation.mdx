# 变更操作

本文档提供 ObjectQL 变更操作的完整规范。

## 插入

```typescript
await db.mutation('customer', {
  action: 'insert',
  data: {
    name: 'Acme Corp',
    email: 'contact@acme.com',
    status: 'active'
  }
})
```

## 更新

```typescript
await db.mutation('customer', {
  action: 'update',
  filters: [['_id', '=', customerId]],
  data: {
    status: 'inactive',
    modified: new Date()
  }
})
```

## 删除

```typescript
await db.mutation('customer', {
  action: 'delete',
  filters: [['status', '=', 'archived']]
})
```

## 批量操作

### 批量插入

```typescript
// 批量插入
await db.mutation('product', {
  action: 'insert',
  data: [
    { name: '产品 1', price: 10 },
    { name: '产品 2', price: 20 },
    { name: '产品 3', price: 30 }
  ]
})
```

### 批量更新

```typescript
// 批量更新
await db.mutation('task', {
  action: 'update',
  filters: [['project_id', '=', projectId]],
  data: { status: 'cancelled' }
})
```
